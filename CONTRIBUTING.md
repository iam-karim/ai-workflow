# Contributing

We welcome contributions to the AI-Workflow Engineering OS!

## Principles
1. **No new execution layers.** We do not accept PRs that attempt to turn AI-Workflow into an agent orchestrator.
2. **Minimalism.** If a feature does not provide measurable long-term value, it will be rejected.
3. **AI Agnostic.** Features must be useful for software engineering generally, not just for a specific AI vendor.

## Setup
1. Clone the repository.
2. Run `pnpm install`.
3. The monorepo packages are located in `packages/`.
4. Ensure you run `pnpm run test` and `pnpm run lint` before submitting a PR.
