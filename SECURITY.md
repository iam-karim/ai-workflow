# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within AI-Workflow, please send an e-mail to security@ai-workflow.io. All security vulnerabilities will be promptly addressed.
